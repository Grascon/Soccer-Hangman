Welcome to Soccer Hangman!

This site was built using HTML, CSS, JavaScript, jQuery, Bootstrap 4, Google Fonts, and Font Awesome. In order to play, press any letter key to begin guessing which soccer player it is. The chosen player is selected randomly and you have 10 guesses before you lose. The game will then restart. Good luck!
